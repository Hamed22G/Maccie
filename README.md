# Mijn Diensten 🍟

McDonald's shift tracker – werkt als een echte app op je telefoon.

## Hoe zet je het online (GitHub Pages)

### Stap 1 – Maak een GitHub account
Als je die nog niet hebt: ga naar [github.com](https://github.com) en maak een gratis account.

### Stap 2 – Maak een nieuwe repository
1. Klik op het **+** icoontje rechtsboven → **New repository**
2. Naam: `mijn-diensten` (of wat je wilt)
3. Zet het op **Public**
4. Klik **Create repository**

### Stap 3 – Upload de bestanden
1. Klik op **uploading an existing file**
2. Sleep alle bestanden hierin:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.png`
3. Klik **Commit changes**

### Stap 4 – Zet GitHub Pages aan
1. Ga naar **Settings** van je repository
2. Scroll naar **Pages** in het linkermenu
3. Onder **Source** → kies **main** branch
4. Klik **Save**
5. Na 1-2 minuten krijg je een link zoals: `https://jouwusername.github.io/mijn-diensten`

### Stap 5 – Voeg toe aan je telefoon

**iPhone (Safari):**
1. Open de link in Safari
2. Tik op het Deel-icoontje (vierkantje met pijl omhoog)
3. Tik op **Zet op beginscherm**
4. Klaar! 🎉

**Android (Chrome):**
1. Open de link in Chrome
2. Tik op de 3 puntjes rechtsboven
3. Tik op **Toevoegen aan startscherm**
4. Klaar! 🎉

---

De app werkt ook offline als je eenmaal de pagina hebt geopend.
